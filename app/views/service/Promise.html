ExposedPromise function Angular Promise ExposedPromise Wrapps an Angular Promise and allows a promise to be cancelable. This class combined with SniffedPromise allows the API to check a Promise for its intention and decide whether to allow a promise to continue or not. This is useful in scenarios where Promises are being queued up and can be forgotten/dropped.
