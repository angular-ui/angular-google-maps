uiGmapLogger Service

This logger utilizes $log underneath. The API now logs errors by default with the logger enabled. Therefore the default currentLevel is "error".
source code
.controller("someController", function(uiGmapLogger) { //uiGmapLogger.doLog = true; //no longer necessary, but if you want to disable it set it false or change currentLevel uiGmapLogger.currentLevel = uiGmapLogger.LEVELS.debug; log.info("hmm"); //not logged log.debug("huh!"); //logged log.warn("uhoh!"); //logged log.error("oh crap!"); //logged });
