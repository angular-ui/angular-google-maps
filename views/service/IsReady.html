This service lets you know when ui-gmap is ready. As uiGmapIsReady passes a promise callback of maps. Beyond that, the instances passed back to you come back with the following info:
  • instance: map instance number (internal counter for ui-gmap on which map)
  • map: The actual gMap (google map sdk instance of the google.maps.Map).
    map.uiGmap_id: A unique UUID that gets assigned to the gMap via ui-gamp api.
The uiGmapGoogleMapApi service should not be used to wait for directives to finish augmenting control objects. uiGmapIsReady (this service) should be used instead.

.controller("someController", function(uiGmapIsReady) { uiGmapIsReady.promise(1).then(function(instances) { instances.forEach(function(inst) { var map = inst.map; var uuid = map.uiGmap_id; var mapInstanceNumber = inst.instance; // Starts at 1. }); }); });
